How it works:

  1. Sign in with Google using your TakingITGlobal email address
  2. Complete your profile setup with Airtable API key and ServiceNow credentials
  3. Review sessions that have "Booked" status in Airtable
  4. Start the automation and monitor progress in real-time
  5. Review results and check updated Airtable records

Prerequisites:

  • TakingITGlobal Google account for authentication
  • Airtable API key with access to Sessions table
  • ServiceNow account credentials and 2FA setup
  • Sessions must have "Booked" status in Airtable
  • Zoom meetings should be created externally and linked in the "WebEx/Zoom Link" field

Security Note:

This app uses secure OAuth 2.0 authentication. No passwords are stored in the app.
